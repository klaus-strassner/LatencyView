Sub-millisecond analysis for NVIDIA Reflex.

01

Hardware
  • Connect the monitor upstream cable to the computer.
  • Connect the mouse to the designated Reflex port on the monitor.
  • Enable Reflex Latency Analyzer in the monitor OSD.
Compatible Hardware ↗
02

Software
  • Install FrameView and the NVIDIA App.
  • Define the output directory and assign the benchmark hotkey.
  • Enable NVIDIA Reflex inside the game settings.
  • Align the OSD indicator and run the benchmark.
03

Automation
  • Create a macro to automate mouse clicks during a benchmark run.
