Initialization

Configure hardware and software to capture precise telemetry data.

01

Hardware
  • Connect the monitor upstream cable to the computer.
  • Connect the mouse to the designated Reflex port on the monitor.
  • Enable Reflex Latency Analyzer and Flash Indicator in the monitor OSD.
View Compatible Hardware ↗
02

Software
  • Install the NVIDIA App and enable the overlay.
  • Define the output directory and assign a benchmark key in the performance settings.
  • Enable NVIDIA Reflex inside the game settings.
  • Align the OSD indicator over the weapon flash. Execute the benchmark.
Download NVIDIA App ↗
03

Automation
  • Map a single mouse button to execute the benchmark key and a primary click simultaneously using peripheral software.
